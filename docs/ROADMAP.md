## Bigger workstreams (planned)

- **Tectonic feature graph** (`docs/research/tectonic-feature-graph.md`, user proposal June 2026 ‚Äî **ACTIVE: T0 shipped v0.058**): the largest realism gap ‚Äî ranges are currently `stress ‚Üí blur ‚Üí height` (smooth blobs). Add the missing layer between plate interaction and height: tangential **shear** field, **boundary-type matrix** (crust A√óB √ó convergence √ó shear), boundary **polyline graph**, and feature synthesis along segments (multi-ridge orogenic kernel, fold belts `sin(k‚àö|C|¬∑d)`, trench+arc, foreland basin, transform offsets). ~6 phased versions (T0 shear/metadata ‚Üí T1 graph ‚Üí T2 uplift kernel ‚Üí T3 boundary-type features ‚Üí T4 transforms ‚Üí T5 tuning), each opt-in/gated + bit-identical at defaults + headless-verifiable. Reuses `distanceToBoundary`, `applyFeatureAlongCurve` (distance-field stamp), `rdpSimplify`, and the existing erosion. Independent of the asset/visual work; world-res only (16k tiles inherit it via `amplifyRegion`). Refs: Cortial 2019, Cordonnier 2016.
